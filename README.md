Aplikacja służy do rezerwowania i wypożyczania książek z biblioteki. 
Klient może zarezerwować książkę i anulować rezerwację, może raz przedłużyć wypożyczenie książki o miesiąc oraz przeglądać i wyszukiwać dostępne. 
Użytkownik z rolą bibliotekarza zatwierdza wypożyczenia, przedłuża terminy, zaznacza zwroty i nalicza kary za przekroczenie terminu zwrotu.
Administrator może dodatkowo wykonywać operacje CRUD na zasobach takich jak książki, kategorie, autorzy, wydawcy.
